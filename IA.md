O objetivo não foi substituir o raciocínio técnico, mas sim melhorar clareza, produtividade e qualidade da entrega.

## 3. Criação de migrations e dados iniciais para testes

### Prompt utilizado

Me ajude a criar migrations e seeds para um projeto Node.js com TypeScript focado em um fluxo simples de checkout.

Preciso de dados iniciais para testes, incluindo:

- Usuários padrão;
- Produtos focados em tecnologia;
- Capas de celular;
- Preços;
- Quantidade em estoque;
- Links de imagens reais ou simuladas para os produtos.

Os produtos devem parecer realistas para uma loja chamada CaseCellShop.  
Gere exemplos de produtos com nomes, descrições, preços, estoque e URLs de imagem.

### Objetivo do uso

Usei este prompt para acelerar a criação de dados iniciais do projeto.  
A ideia foi ter uma base simples para testar o checkout sem precisar cadastrar tudo manualmente.

## 4. Integração do front-end com o back-end

### Prompt utilizado

Tenho um back-end em Node.js com TypeScript e um front-end em React com TypeScript.

Me ajude a organizar o consumo das APIs do back-end no front-end.  
Preciso criar funções para:

- Buscar produtos;
- Criar uma tentativa de checkout;
- Tratar estados de carregamento;
- Tratar erros de validação;
- Exibir mensagens amigáveis para o usuário;
- Evitar múltiplos cliques durante o processamento da compra.

Quero uma solução simples, organizada e fácil de entender para um desafio técnico júnior full stack.

### Objetivo do uso

Usei este prompt para otimizar o tempo de integração entre front-end e back-end.  
A IA ajudou a estruturar melhor o consumo das APIs, os estados de loading e o tratamento de erros no fluxo de compra.

## 5. Melhoria da experiência e estilização do front-end

### Prompt utilizado

Estou criando uma tela simples de checkout em React com TypeScript para um desafio técnico.

Me ajude a melhorar a estilização da interface mantendo simplicidade.  
A tela precisa mostrar produtos, permitir selecionar quantidade, iniciar compra, bloquear ações duplicadas durante o carregamento e exibir mensagens de sucesso ou erro.

Sugira melhorias de layout, espaçamento, organização visual e feedback para o usuário, sem deixar o projeto complexo demais.

### Objetivo do uso

Usei este prompt para melhorar a aparência e usabilidade do front-end.  
A IA ajudou com sugestões de organização visual, feedback de carregamento e mensagens mais claras para o usuário.
